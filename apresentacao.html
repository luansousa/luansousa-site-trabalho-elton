API Restful - SpringBoot

Através do mapeamento de URL e dos verbos HTTP, é possível fazer requisições para uma aplicação Java rodando em um servidor e receber as informações dos veículos do banco de dados. Dessa forma, o algoritmo de OCR irá decodificar a placa do veículo e fazer uma requisição HTTP para a API, e depois, através da interface gráfica, mostrar se o veículo está autorizado ou não a entrar no condomínio.
